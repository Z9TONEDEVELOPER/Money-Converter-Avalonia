# Money Converter - Учебный проект на Avalonia UI

Этот проект представляет собой простое приложение-конвертер валют, разработанное с использованием фреймворка Avalonia UI. Проект создан в учебных целях и демонстрирует ключевые концепции современной разработки desktop-приложений на .NET.

## Особенности проекта

- **INotifyPropertyChanged**: Реализация интерфейса `INotifyPropertyChanged` для обеспечения двусторонней привязки данных между интерфейсом и моделью данных
- **Работа с API**: Приложение получает актуальные курсы валют из публичного REST API (open.er-api.com)
- **Обработка JSON**: Десериализация JSON-ответов от API с использованием `System.Text.Json`
- **Асинхронное программирование**: Использование async/await для работы с сетевыми запросами без блокировки UI
- **Data Binding**: Привязка данных между XAML-интерфейсом и C#-кодом через свойства ViewModel

## Функциональность

- Выбор исходной и целевой валют из выпадающих списков
- Конвертация суммы между валютами
- Возможность поменять местами валюты кнопкой "↔"
- Отображение статуса загрузки курсов валют
- Обработка ошибок сети и валидация ввода

## Технологии

- Avalonia UI (кроссплатформенный фреймворк для desktop-приложений)
- C# 10+
- .NET 6
- HttpClient для работы с REST API
- System.Text.Json для обработки JSON

## Установка и запуск

```bash
# Клонировать репозиторий
git clone https://github.com/yourusername/money-converter.git

cd money-converter

# Восстановить зависимости
dotnet restore

# Собрать проект
dotnet build

# Запустить приложение
dotnet run
```

## Цель проекта

Этот учебный проект предназначен для демонстрации:
- Основ работы с Avalonia UI
- Практического применения `INotifyPropertyChanged` 
- Работы с внешними API и обработки JSON-данных
- Асинхронного программирования в desktop-приложениях
- Паттерна MVVM в реальном сценарии

Проект может быть использован как стартовая точка для изучения кроссплатформенной разработки desktop-приложений на .NET.
